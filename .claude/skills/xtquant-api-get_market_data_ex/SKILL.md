---
name: 获取股票k线数据
description: 本技能用于使用xtquant库(原生python接口)的xtdata模块查询本地历史k线或实时K线，支持多种时间周期，周期包括日线(1d)、1分钟(1m)和5分钟(5m)和tick 15m、30m、1h（1小时周期） 1w(周线) 1mon(月线)等。
提供的k线字段包括：开、高、低、收、成交额、成交量、前收等
---

For better usage,see [REFERENCE.md]. For examples, see [EXAMPLES.md].

Run the helper script:

python scripts/get_market_data_ex.py input.txt

> tips
> 取期货品种的行情时，建议fill_data始终传False
